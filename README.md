Smart Campus Finder is a user-friendly web application designed to help students and visitors quickly find essential facilities for campus environment.

The system provides categorized sections such as Food, Rooms, Hospitals, Police Stations, and Bus Stops. Each section displays important details like name, contact information, and location.

The application integrates map functionality using Leaflet.js to show real-time locations with latitude and longitude coordinates, making navigation simple and efficient.

 Features:
- Interactive dashboard with categorized service cards
- Real-time location mapping using Leaflet
- Clean and responsive UI using Tailwind CSS
- Search functionality for quick access
- Organized layout for better user experience

 Technologies Used:
- Frontend: HTML, CSS, Tailwind CSS, JavaScript
- Map Integration: Leaflet.js
- Backend (Planned): Spring Boot
- Database (Planned): MySQL

 Purpose:
The main goal of this project is to simplify campus navigation and improve accessibility to essential services for students and visitors.

- User login and personalized features
- Live location tracking
